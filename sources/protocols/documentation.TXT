===========>Справочник для стандартного и нестандартного протокола

[ABB] - Аббревиатура
[YEAR] - Год
[ION_NAME] - Имя иона
[ION_NUM] - Номер иона
[PRTCL_NUM] - Номер протокола
[SESSION_NUM] - Номер сеанса
[BENCH] - Станок
[ORG_NAME] - Наименование организации
[CIPER] - Шифр или наименование работы
[IRR_PRODUCT] - Облучаемое изделие
[START_DATA] - Дата начала облучения
[START_TIME] - Время начала облучения
[DURATION] - Длительность облучения
[EXP_CONDITION] - Условия эксперемента
[ANGLE] - Угол
[TEMP] - Температура,°C
[DEG_MATERIAL] - Материал дегрэйдора
[THICKNESS] - Толщина, мкм
[E_VALUE] - Энергия Е на поверхности, МэВ/н
[E_MEASURE] - Погрешность измерения энергии
[MILEAGE_VALUE] - Пробег, R [Si],мкм
[MILEAGE_MEASURE] - Погрешность измерения пробега
[LINE_LOSS_VALUE] - Линейные потери энергии ЛПЭ, МэВ×см2/мг [Si]
[LINE_LOSS_MEASURE] - Погрешность потери энергии
[K_VALUE] - Значение K
[K_MEASURE] - Погрешность K
[COUNTER_1_DATA] - Данные 1 счётчика
[COUNTER_2_DATA] - Данные 2 счётчика
[COUNTER_3_DATA] - Данные 3 счётчика
[COUNTER_4_DATA] - Данные 4 счётчика
[AVG_DATA_VALUE] - Средние значения по счётчикам
[ADM_PRTCL_NUM] - Номер протокола доступа
[ACTUAL_KOEF] - Фактический коэфициент K
[DTCTR_1_DATA] - Данные 1 детектора
[DTCTR_2_DATA] - Данные 2 детектора
[DTCTR_3_DATA] - Данные 3 детектора
[DTCTR_4_DATA] - Данные 4 детектора
[DTCTR_5_DATA] - Данные 5 детектора
[DTCTR_6_DATA] - Данные 6 детектора
[HTRGNT] - Неоднородность
[HTRGNT_LEFT] - Неоднородность по левому
[HTRGNT_RIGHT] - Неоднородность по правому
